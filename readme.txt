#我的野营炉
	2021.01.13
	设计需求：
		1.燃料充分燃烧，尽量少产生烟雾和CO
		2.能够燃烧木柴和燃料颗粒
		3.方便清理炉灰
		4.可以做饭，也可以放在帐篷内取暖，取暖时表面温度不能太高，否则可能引起火灾
		5.预留烧烤格子，至少可以烤一个红薯<br>
		6.考虑添加水冷接口，用水冷设备群暖，水量小于4L
		7.嵌入一个温度表显示温度
		8.考虑烟囱尽量少带走热量
